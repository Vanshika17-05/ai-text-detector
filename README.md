
# ⚡ AI Text Detector

An interactive web-based application that analyzes text and predicts whether it is AI-generated or human-written using linguistic pattern analysis and probability scoring.

---

## 🚀 Features

- 🔍 Real-time text classification (AI vs Human)
- 📊 Confidence score and probability breakdown
- 📈 Interactive charts (Chart.js)
- 🧠 Linguistic feature analysis (sentence length, vocabulary richness, etc.)
- 🌓 Dark/Light mode toggle
- 🗂️ History tracking of previous analyses
- ⚙️ Customizable detection sensitivity

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Visualization: Chart.js  
- Logic: Heuristic-based NLP feature analysis

## 📂 How It Works

The system analyzes input text based on:
- Sentence structure and length  
- Vocabulary diversity (Type-Token Ratio)  
- Word complexity and frequency  
- Presence of formal vs informal language patterns  

These features are combined to generate:
- AI Probability Score  
- Human Probability Score  
- Confidence Level  

---

## ⚠️ Note

This version uses heuristic-based analysis and is designed for demonstration and educational purposes. It simulates machine learning behavior but does not use a trained ML model.
